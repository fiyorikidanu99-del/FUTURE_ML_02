# Support Ticket Classification System (Task 2 - Machine Learning Internship)

## Objective
Develop a machine learning model that automatically classifies customer support tickets into different categories and assigns priority levels.

## Features
- Text preprocessing using TF-IDF
- Ticket classification using Multinomial Naive Bayes
- Priority assignment (High, Medium, Low)
- Model evaluation with accuracy and classification report

## Technologies Used
- Python
- Pandas
- Scikit-learn
- TF-IDF Vectorizer
- Jupyter Notebook

## Sample Categories
- Network
- Hardware
- Software
- Account
- Email

- Natural Language Processing (NLP)
- Text Classification
